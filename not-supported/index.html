対応ブラウザについて

System Requirements

お使いのブラウザは、当ホームページではサポートしておりません。

やさしいロボコンの公式ホームページは、以下のブラウザでの閲覧を推奨しています。

  • Google Chrome
  • Microsoft Edge
  • Apple Safari

また、以下のブラウザでも、一部表示に差異が生じる可能性はありますが動作を確認しております。

  • Mozilla Firefox

上記のブラウザ以外では、コンテンツが正しく表示されない場合があります。

Internet Explorerにつきましては、Microsoft社のサポート終了を鑑みて、当ホームページでは対応しておりません。また、今後も対応の予定はございません。

推奨のブラウザでホームページにアクセスいただきますようお願いいたします。

また、JavaScriptにつきましては有効にしていただきますようお願いいたします。
